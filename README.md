## ğŸ’¡ Sample Code

**Binomial Probability Calculation:**
```r
# P(7 <= X <= 9) where X ~ Bin(12, 1/6)
p_bino <- pbinom(9, 12, 1/6) - pbinom(6, 12, 1/6)
cat("P(7 <= X <= 9):", p_bino, "\n")
```

**Plotting a t-distribution:**
```r
set.seed(123)
hist(rt(100, 10), main="t-distribution, df=10")
```
